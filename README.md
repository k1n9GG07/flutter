# Static Resources App

Flutter-приложение, демонстрирующее работу со статическими ресурсами: изображениями и пользовательскими шрифтами.

##  Функциональность

- **Галерея изображений**: Отображение трех растровых изображений из ассетов.
- **Динамический Layout**: Возможность переключения между вертикальным списком и горизонтальным рядом.
- **Кастомный шрифт**: Использование шрифта `CustomFont` (Courier) во всем приложении через глобальную тему.
- **Material 3**: Современный интерфейс с использованием дизайн-системы Material 3.

##  Технологии

- **Flutter**: 3.0.0+
- **Dart**: 3.0.0+
- **Assets**: PNG изображения и TTF шрифты.

##  Структура проекта

```text
static_resources_app/
├── assets/
│   ├── images/       # Графические ресурсы
│   └── fonts/        # Пользовательские шрифты
├── lib/
│   └── main.dart     # Основная логика UI
└── pubspec.yaml      # Конфигурация ресурсов и зависимостей
```

##  Установка и запуск

1. **Клонируйте репозиторий**:
   ```bash
   git clone https://github.com/k1n9GG07/flutter.git
   cd flutter/static_resources_app
   ```

2. **Загрузите зависимости**:
   ```bash
   flutter pub get
   ```

3. **Запустите приложение**:
   ```bash
   flutter run
   ```
